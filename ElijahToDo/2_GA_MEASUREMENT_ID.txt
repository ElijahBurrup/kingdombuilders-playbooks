ACTION ITEM 2: Set Up Google Analytics
=======================================
Priority: HIGH (do this first -- once done, Claude can verify it's tracking)
Time: ~5 minutes

WHY: You need this BEFORE running ads. Google Ads and Meta Ads both need
analytics to track conversions (did the person who clicked the ad actually buy?).

STEPS:

1. Go to https://analytics.google.com
2. Sign in with burrupgpt@gmail.com
3. Click the gear icon (Admin) at bottom-left
4. Click "Create" -> "Property"
   - Property name: Kingdom Builders Playbooks
   - Time zone: your time zone
   - Currency: USD
5. Click "Next" -> select "Business" -> click "Create"
6. Under "Data Streams", click "Web"
   - Website URL: https://kingdombuilders.ai
   - Stream name: Playbooks
   - Click "Create stream"
7. You'll see a Measurement ID that looks like: G-XXXXXXXXXX
   Copy it.
8. Go to Render Dashboard:
   https://dashboard.render.com
   -> kb-playbooks service -> Environment -> Add Environment Variable
   Key:   GA_MEASUREMENT_ID
   Value: G-XXXXXXXXXX (paste your actual ID)
9. Click "Save Changes" (Render will auto-redeploy)

ONCE YOU'VE DONE THIS:
Tell Claude "I set up Google Analytics, verify it works" and Claude can check
that the tracking code is loading on the live site.

NEXT STEP (when ready for ads):
You'll connect this GA4 property to Google Ads for conversion tracking.
That's a separate setup but uses this same Measurement ID.
